PurpleOrange

A social network experiment, developed by the group IKOM.

For now, still in alpha testing, this social network features:

  • radio_button_checked Friends system (profiles, requests, chat)
  • radio_button_checked Search system
  • radio_button_checked Publications (without and with images)
  • radio_button_checked Top navigation bar and sidebar
  • radio_button_checked Mobile compatible
  • radio_button_checked Side bar becomes the default navigation method when in mobile

The website is currently open to public, but it is planned to go private upon starting the Beta phase.
When and if that happens, a demo version of the site will be available to check out the functionalities. For now, you can access the website below.
